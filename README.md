# Api simples feita usando node js


## Rotas

### Get
http://localhost:3000/customers

### Get por id
http://localhost:3000/customers/:id

### Post
http://localhost:3000/customers

### Update
http://localhost:3000/customers/:id

### Delete
http://localhost:3000/customers/:id



## Para executar o projeto em sua máquina clone o repositório e execute o comando `npm i`
